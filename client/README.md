# Async Race

## Getting Started

This project consists of a React frontend and a backend API. Both services must be running for the application to work correctly.

### Frontend

From the project root directory:

```bash
npm install
npm start
```

The application will be available at:

```
http://localhost:3000
```

### Running Tests

```bash
npm test
```

### Backend

Navigate to the API directory:

```bash
cd server/async-race-api
```

Install dependencies and start the server:

```bash
npm install
npm start
```

## Building for Production

```bash
npm run build
```

This creates an optimized production build in the `build` directory.
